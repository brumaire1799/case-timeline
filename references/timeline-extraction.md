# 法律案件时间轴提取指南

从案件材料中准确提取时间节点、事件描述和归属方的参考。

---

## 归属方分类规则（party 字段）

提取每个事件时，必须标注归属方。归属方决定时间轴上的位置和颜色。

### 多方当事人场景

案件可能涉及多于两方当事人（如原告一、原告二；被告一、被告二、被上诉人三等），此时每方独立标注。最终位置（上下方）和颜色将在第二步由用户确认。

### 归属方判断规则

- **合同签订/协议签署**：按参与方分别标注，或合并为一方
- **单方行为**（付款、交付、通知、催告、违约等）：按行为主体判断
- **法院行为**（立案、开庭、判决、送达等）：标注为具体法院或"法院"
- **沟通/协商**：如双方均有参与则标注双方，如仅为一方发函/回复则为该方
- **第三方事件**（鉴定、评估、公证等）：标注为具体机构或"第三方"
- **多方案件**：同一方当事人可能有多名（如"原告一""原告二"），需逐一区分

### 提取时的 party 值建议

在提取阶段，可使用以下命名规则方便后续映射：

| party 值示例 | 含义 | 建议最终位置 | 建议颜色 |
|-------------|------|-------------|---------|
| `原告` / `原告一` / `原告二` | 原告方各当事人 | 上方 | 暖色系（红/橙/金） |
| `被告` / `被告一` / `被告二` | 被告方各当事人 | 下方 | 冷色/暗色系（黑/蓝/绿） |
| `上诉人` / `被上诉人` | 上诉场景各方 | 上方/下方 | 同上 |
| `法院` | 法院立案/开庭/判决等 | 下方 | 灰色 |
| `第三方` | 鉴定机构、评估机构等 | 下方 | 灰色 |

## 时间格式规范

- **标准格式**：`YYYY年MM月DD日`
- **不精确日期**：如只有年份，写 `2024年`；只有年月，写 `2024年3月`
- **时间段**：如"2024年1月至3月"，拆为起止两个节点
- **相对时间**：如"签约后第15天"，尽量推算具体日期；无法推算则保留相对描述并标注
- **跨年密集事件**：可在 `extra_label` 字段统一标注年份（如 `均为2021年`），日期仅保留月日（如 `9月8日`），建议用户选择是否简化

## 从不同材料中提取

### 判决书/裁定书

判决书的"经审理查明"部分是时间节点的核心来源。关注：
- "XXXX年XX月XX日，原告与被告签订……"
- "同年X月X日" → 结合上文确定年份
- "次日"/"翌日" → 推算为具体日期
- 判决主文中的时间（立案日期、开庭日期、判决日期）也需提取
- **多方案件注意**：判决书可能列出原告一、原告二等多方，需分别标注

### 起诉状/答辩状

按"事实与理由"部分的时间叙述提取。注意双方当事人对同一事件的日期可能存在分歧，标注来源。
多方案件中，注意区分不同原告/被告的独立行为。

### 合同

提取关键日期：
- 签订日期
- 各期履行期限
- 违约时间点
- 通知/催告日期
- 解除/终止日期

### 证据材料

- **转账记录**：提取转账日期、金额、收付款方
- **聊天记录**：提取消息日期时间、关键对话内容
- **邮件**：提取发送/接收日期、主题、关键内容
- **微信/钉钉截图**：OCR或截图中提取时间戳和关键内容

### 律师口述

律师口述通常较口语化，需：
1. 逐条记录律师描述的事件
2. 追问每个事件的具体日期
3. 将口语转化为规范的事件描述（一句话概括，动作+客体+结果）
4. 多方案件需逐一确认每方当事人的准确名称和角色
5. 提交确认前再次核对

---

## 事件描述规范

- **动作导向**：以核心法律行为/事实为主语或谓语
- **简洁原则**：每条事件控制在一句话内，尽量不超过40字
- **客观中性**：不添加主观法律评价（如"恶意"等形容词保留为客观描述）
- **支持多行**：超过文本框宽度自动换行，无需刻意压缩

示例：
- ❌ "被告恶意拖欠货款，严重违约"
- ✅ "被告逾期未支付货款"
- ✅ "原告向被告发出书面催告函"
- ✅ "双方签订《供货合同》，约定货款支付方式为月结30天"

---

## 确认清单

在向用户提交确认前，检查：
1. 所有日期格式是否统一
2. 每个事件是否都标注了 party
3. 事件是否按时间顺序排列
4. 描述是否简洁、客观
5. 是否有明显遗漏的关键节点（参考常见案件类型的节点清单）
6. **多方案件**：每方当事人的名称是否一致（避免同一方出现"原告一"/"原告1"混用）

## 常见案件类型的时间轴重点

### 合同纠纷
合同签订 → 履行节点 → 违约发生 → 催告/协商 → 起诉/仲裁 → 判决

### 劳动争议
入职 → 关键事件 → 争议发生 → 劳动仲裁 → 法院起诉 → 判决

### 婚姻家庭
结婚 → 子女出生 → 财产变动 → 分居 → 起诉离婚 → 判决

### 刑事案件
案发 → 报案 → 立案 → 拘留/逮捕 → 审查起诉 → 一审 → 二审
