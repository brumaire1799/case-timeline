# 多方当事人配色参考

当案件涉及多于两方当事人时（如原告一、原告二、被告一、被告二、上诉人、被上诉人等），每方配置独立颜色，以便在时间轴上一目了然地区分。

---

## 预设调色板

### 上方位置推荐色（暖色系）

适用于希望突出显示的当事人（通常为主动方/原告方/上诉方）：

| 顺序 | 色号 | 名称 | 适用场景 |
|------|------|------|---------|
| 1 | `#CC0000` | 正红 | 默认上方第一方（原告/上诉人/申请人） |
| 2 | `#E85D3F` | 珊瑚红 | 上方第二方（如原告二） |
| 3 | `#FF8C00` | 深橙 | 上方第三方（如原告三） |
| 4 | `#B8860B` | 暗金 | 上方第四方 |

### 下方位置推荐色（冷色/暗色系）

适用于时间轴下方的当事人（通常为被动方/被告方/被上诉方）：

| 顺序 | 色号 | 名称 | 适用场景 |
|------|------|------|---------|
| 1 | `#333333` | 炭黑 | 默认下方第一方（被告/被上诉人/被申请人） |
| 2 | `#1A5276` | 深蓝 | 下方第二方（如被告二） |
| 3 | `#2E7D32` | 深绿 | 下方第三方（如被告三） |
| 4 | `#6C3483` | 深紫 | 下方第四方 |

### 中立/第三方推荐色

适用于法院、仲裁机构、鉴定机构等中立主体：

| 顺序 | 色号 | 名称 | 适用场景 |
|------|------|------|---------|
| 1 | `#666666` | 中灰 | 法院行为 |
| 2 | `#888888` | 浅灰 | 第三方机构（鉴定、评估） |
| 3 | `#8B4513` | 棕色 | 案外关联方 |

---

## 颜色可访问性

- 所有预设颜色在白色背景上均满足 WCAG AA 级对比度要求（≥4.5:1）
- 深色背景（投屏）下建议使用较亮的颜色变体
- 如需自定义颜色，推荐使用 [WebAIM Contrast Checker](https://webaim.org/resources/contrastchecker/) 验证对比度

---

## 使用示例

### 两方场景（默认）

```json
{
  "parties": [
    {"name": "原告", "color": "#CC0000", "position": "top"},
    {"name": "被告", "color": "#333333", "position": "bottom"}
  ]
}
```

### 多方场景（原告方两人 + 被告方三人 + 法院）

```json
{
  "parties": [
    {"name": "原告一", "color": "#CC0000", "position": "top"},
    {"name": "原告二", "color": "#E85D3F", "position": "top"},
    {"name": "被告一", "color": "#333333", "position": "bottom"},
    {"name": "被告二", "color": "#1A5276", "position": "bottom"},
    {"name": "被上诉人三", "color": "#2E7D32", "position": "bottom"},
    {"name": "法院", "color": "#666666", "position": "bottom"}
  ]
}
```

### 混合位置场景（某一原告方在时间轴下方）

有时同一角色的不同当事人可能分处上下方（如原告一在上，但原告二因事件性质被安排在下方）。通过 `position` 字段可灵活配置：

```json
{
  "parties": [
    {"name": "原告一", "color": "#CC0000", "position": "top"},
    {"name": "原告二", "color": "#E85D3F", "position": "bottom"},
    {"name": "被告一", "color": "#333333", "position": "bottom"},
    {"name": "被告二", "color": "#1A5276", "position": "top"}
  ]
}
```

## 颜色分配原则

1. **同一侧的颜色应保持色相区分度**：避免上方同时使用两个相近的红色
2. **上下方颜色应有明显的冷暖/明暗区分**：确保投屏时上下方一目了然
3. **法院/第三方建议使用灰色调**：体现中立性
4. **颜色不超过 6-7 种**：过多颜色会降低可读性，如当事人超过 7 方，建议合并关联方或使用同一颜色加编号区分
